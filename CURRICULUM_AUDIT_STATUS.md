## 🎓 PEDAGOGICAL NOTES

### Quiz Design Principles
- Mix difficulty: 1 recall + 1 application question minimum
- Options should be plausible distractors (not obviously wrong)
- Explanations should teach, not just confirm
- Link back to lesson key concepts

### Real Example Best Practices
- Use recognizable brands (Airbnb, Spotify, Apple)
- Explain the "why" behind the design decision
- Connect to lesson principles explicitly
- Optional: Good vs Bad comparison

### AI Prompt Quality
- Provide context: "I'm designing [X] for [audience]"
- Be specific: Include paste instructions [paste list]
- Output format: "Generate 5..." not "Help me with..."
- Diverse use cases: synthesis, ideation, validation
